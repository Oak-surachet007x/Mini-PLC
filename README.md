# Mini-PLC

## introduction :memo::speech_balloon:
- Arduino uno R4 ในการควบคุมระบบสายพานและส่งข้อมูลต่างๆให้กับ Node-red
- Node-red รับค่าข้อมูลมาแสดงผ่าน Dashboard 
- Arduino กับ Node-red สื่อสารด้วย Mqtt Broker
